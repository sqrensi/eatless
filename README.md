# Бот «Меньше есть»

Телеграм-бот для осознанного питания: чек-лист перед едой, секундомер приёма пищи, правило «подожди 10 минут» перед перекусом, учёт ккалорий и лимит в день.

## Запуск локально

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
# Linux/macOS: source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

Токен бота задаётся в файле `.env`:

```
BOT_TOKEN=твой_токен_от_BotFather
```

## Запуск на сервере Linux

**Полная пошаговая инструкция:** [docs/DEPLOY.md](docs/DEPLOY.md) — от подключения по SSH до systemd и обновления.

Кратко:

1. Подключись по SSH, установи Python 3.10+, git.
2. Склонируй репозиторий или скопируй проект в `~/bot`.
3. `python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt`
4. Создай `.env` с `BOT_TOKEN=...`, при необходимости выставь часовой пояс.
5. Запуск через systemd: `deploy/kk-bot.service` скопировать в `/etc/systemd/system/`, поправить пути и пользователя, затем `systemctl enable --now kk-bot`.

## Возможности

- **Собираюсь поесть** — чек-лист (вода, голод, тарелка), выбор завтрак/обед/ужин, секундомер, напоминания попить воды; по окончании — вопрос про 80% сытости и запись ккалорий.
- **Хочу перекусить** — ожидание 10 минут, затем «Всё ещё хочется?»; запись перекуса с ккалориями.
- **Мой прогресс** — ккалории за день, лимит 1700, время за едой, стрики.
- Граница **не есть после 21:00** (по времени сервера).

Данные хранятся в SQLite (`bot_data.db`).
